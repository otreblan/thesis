\customchapter{ABSTRACT}

Currently, CUDA is the most popular compute API for GPUs.
But it is limited to run only on Nvidia hardware.
Propietary APIs exists, but using them would mean rewriting the same logic for each target.
On the other hand, Vulkan allows a myriad of GPUs from different vendors to use a single API.
Fdtd Lucuma uses it to accelerate the finite-difference time-domain (FDTD) while not being limited to a single GPU family.
